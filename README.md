# Kulture.fm

[kulturefm.slicvic.com](http://kulturefm.slicvic.com)

A free internet radio that plays music from around the world. Built with love using:
  - Vue/Vuex
  - ES6
  - CSS
  - Boostrap 4
  - HTML5
  - SoundCloud API
  - OpenLayers
  - [Restcountries.eu API](https://restcountries.eu)
  - Webpack
  - Lodash
  - Firebase
  - NPM
  - Babel
